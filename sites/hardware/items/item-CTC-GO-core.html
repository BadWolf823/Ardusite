ARDUINO CTC GO! - ОСНОВНОЙ МОДУЛЬ

Что включает в себя каждый модуль?

CTC GO! комплект, который включает в себя все материалы, необходимые для обучения 24 учеников в течение 20 занятий по 45 минут каждый:

  • 8 программируемых плат Arduino UNO WiFi Rev2: с поддержкой WiFi, с возможностью считывания входов и управления выходами
  • 8 Щиты Arduino Education: дополнительные доски, предназначенные для образовательных целей, позволяющие быстро и легко учиться при создании проектов. Эти платы подключаются к программируемым платам для расширения их функциональности и имеют устройство для чтения карт SD и аудиоразъем
  • Два макета разных размеров для каждой платы: используются для упрощения сборки цепей - электронные компоненты, включая резисторы, светодиоды, кнопки и зуммеры
  • Датчики, включая потенциометры, датчики света и ультразвука
  • Восемь 9-вольтовых батарей с двумя разъемами для батарей в каждом - перемычки и USB-кабели
  • Модульные строительные элементы для создания восьми различных управляемых проектов
  • Компонентные блоки для каждой группы студентов со всем необходимым для работы на экспериментальных уроках
  • Запасные компоненты и запасные части для управления учебным классом. Программная платформа для преподавателей со всеми материалами, необходимыми для каждого урока, ресурсы для подготовки к уроку, советы по содержанию, рекомендации по срокам управления учебным классом и ссылки на учебные программы. ученики с пошаговыми инструкциями, видеоматериалами по сборке и веселыми занятиями, которые помогут им начать программирование, электронику и создание полнофункциональных интерактивных проектов (преподаватели также имеют предварительный доступ к этой платформе, чтобы они могли подготовить и адаптировать свои планы уроков ) Обучение и поддержка, включая приветственный обучающий вебинар с экспертом Arduino Education, обучающие видео, в которых разъясняются концепции каждого урока, более короткие видео, раскрывающие содержание урока, и прямую поддержку по электронной почте от эксперта по образованию

Может ли организационная или общественная группа запустить программу?

CTC GO! Программа в первую очередь предназначена для школ, но ее может запустить любая группа, заинтересованная в изучении дисциплин STEAM.

КОТОРЫЙ CTC GO! МОДУЛЬ ВЫ РЕКОМЕНДУЕТЕ?

Если организация, школа или преподаватель впервые покупают CTC GO! модуль CTC GO! Основной модуль - это первый пакет, который должен быть запущен.

Если школа уже прошла этот первый модуль, можно объединить следующие модули так, чтобы они лучше подходили учебному плану, который интересует школа.

Нужно ли покупать еще один учебник или дополнительные материалы для обучения преподавателей?

Нет, преподаватели будут обучаться с использованием тех же материалов, которые их ученики будут использовать позже. Каждый CTC GO! Модуль инструментов содержит все материалы, необходимые для обучения преподавателей.

Каковы минимальные требования в классе?

Школы под управлением CTC GO! Программа должна иметь доступ студентов к компьютерам с Интернетом (по крайней мере, один компьютер для трех студентов) и возможность загружать необходимое программное обеспечение, выделенную комнату, которая может функционировать в качестве мастерской (или, по крайней мере, комнату со столами), и общие аксессуары, такие как как ножницы и наушники.

Как преподаватели могут получить дополнительный доступ к онлайн-платформе?

Пользователи получают доступ к онлайн-платформе в течение года. До 3 преподавателей и 24 ученика получают доступ через учетные записи, которые могут быть активированы только один раз.

Каждый модуль имеет свой собственный код, который даст доступ к его конкретному контенту плюс ко всему предыдущему контенту, где пользователь был ранее зарегистрирован.

Какой у CTC GO! срок службы инструментов? сколько раз я могу использовать набор инструментов?

Средняя продолжительность жизни компонентов, включенных в различные модули CTC GO! около 2 лет.

Когда начинается со отсчет одногодичного доступа к платформе ?

Однолетний доступ к каждому модулю начинается, когда вы впервые активируете его с помощью специального кода (введите ключ продукта). Годовой период не может быть остановлен и возобновлен позже.

Сколько студенческих контактных часов / сессий рекомендуется завершить?

Каждый CTC GO! Модуль состоит из 8 практических занятий, 6 сессий по созданию проектов и 6 экспериментальных сессий, всего 20 сессий. Рекомендуется посвятить 45 - 50 минут за сеанс.

Сколько студентов могут использовать набор инструментов для каждого модуля?

Они включают в себя платы Arduino и щиты EDU (только в модуле Core), компоненты и доступ к платформе для класса до 24 учеников для одновременной работы в группах в рамках проекта. Тем не менее, мы рекомендуем работать в группах из 3 человек.

Сколько Arduino Uno WIFI Boards включает в себя панель инструментов?

В CTC ​​GO есть восемь плат Arduino UNO WiFi! Панель инструментов основного модуля, которая также будет использоваться для следующих модулей.

Какие языки поддерживает онлайн-платформу?

CTC GO! Сама онлайн-платформа в настоящее время доступна на английском языке. Тем не менее, в ближайшем будущем будет добавлено больше языков.

как CTC GO! программа будет включена в учебную программу?

CTC GO! Позволяет работать между учебными планами и включать различные предметы, которые могут быть введены при работе с самой программой. Этот подход обеспечивает идеальное начало для углубления теоретической и практической глубины по широкому кругу предметов. Технологии, математика и физика - предметы для междисциплинарной программы CTC GO! подход, но преподавателям также рекомендуется применять связанные с ним части в других областях желаемой учебной программы.

Это позволяет педагогам включать время по нескольким предметам в своей школе и выводить школу из замкнутой деятельности по предметам, а также в творческую междисциплинарную экосистему.

CTC GO! - Основной модуль приведен в соответствие с Национальной учебной программой для Англии, которая используется в международных школах по всему миру, и NGSS США для предметов STEAM для учащихся в возрасте от 14 до 17 лет. Ссылки на учебные программы предоставляются в программной платформе преподавателей. Кроме того, эти уроки учат студентов важным навыкам 21-го века, таким как сотрудничество, решение проблем, критическое мышление и общение.
