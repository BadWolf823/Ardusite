Инженерный комплект Arduino, версия 2

Arduino Engineering Kit - идеальное решение для университетов и продвинутых студентов, обеспечивающее практическое внедрение технологии Arduino в образовательную среду.

Этот набор предназначен в первую очередь для трех типов пользователей: студентов, изучающих инженерные науки, преподавателей, преподающих инженерные технологии, и тех, кто интересуется инженерными знаниями или имеет опыт работы с ними.

Инженерный комплект включает в себя 3 практических проекта, которые будут охватывать моделирование системы, теорию управления, робототехнику, мехатронику и другие важные инженерные концепции на всех этапах программирования Matlab и Simulink. Эти проекты бросят им интеллектуальный вызов и помогут им развить навыки физической инженерии - плюс они просто забавны.

В комплект входит Arduino Nano 33 IoT, одна из последних профессиональных плат для IoT, выпущенная Arduino. Он также включает в себя несколько индивидуальных деталей, полный набор электроники и механических компонентов, необходимых для сборки всех трех проектов.

В дополнение к высококачественному оборудованию с открытым исходным кодом, после регистрации в Интернете каждый студент получит доступ к специальной платформе электронного обучения и другим учебным материалам. Кроме того, им предоставляется годичная индивидуальная бесплатная пробная лицензия для MATLAB и Simulink. Это дает им практический опыт моделирования систем и разработки встроенных алгоритмов.

Набор продается в прочном пластиковом, складном ящике для инструментов для хранения и многократного использования.

Arduino Engineering Kit построен на собственной системе управления обучением с пошаговыми инструкциями и уроками. Содержимое этого комплекта разделено на шесть глав, включающих краткое введение, руководство по началу работы с инструментами, которые будут использоваться, раздел концепций и, наконец, сами проекты. Пользователи получат доступ к онлайн-платформе в течение одного года и могут приобрести дополнительные лицензии для расширения доступа к платформе.

Описание

В Arduino Engineering Kit Rev2 представлены три практических проекта: самобалансирующийся мотоцикл, робот для рисования и мобильный ровер. Самобалансирующийся мотоцикл: этот мотоцикл может маневрировать на разных территориях и оставаться в вертикальном положении, используя маховик для баланса Мобильный ровер: ровер перемещается между заданными контрольными точками, перемещает объекты с помощью вилочного погрузчика и многое другое Робот для рисования на доске: создайте и запрограммируйте робота, который может дублировать любой чертеж, который он получил, и на доске

В дополнение к оборудованию с открытым исходным кодом, входящим в комплект, каждый студент имеет доступ к платформе электронного обучения с пошаговыми инструкциями, уроки и другие учебные материалы. Студенты также получают годовую бесплатную индивидуальную пробную лицензию для MATLAB и Simulink, предоставляя им практический опыт в системном моделировании и разработке встроенных алгоритмов.

В комплект входят несколько индивидуальных деталей, полный набор электроники и все механические компоненты, необходимые для сборки каждого проекта:

  • Arduino Nano 33 IoT
  • Nano Motor Carrier с IMU и зарядным устройством
  • Три комплекта механических частей для сборки проектов
  • Литий-ионный аккумулятор 18650
  • Два мотор-редуктора с энкодерами
  • Двигатель постоянного тока с датчиками
  • Серводвигатель
  • USB-кабель
  • Два маркера для доски
  • Два колеса
  • Аллен ключ
  • ВЭБ-камера
  • Нейлоновая нить
  • Винты, гайки и болты
  • Прочный пластиковый складной ящик для инструментов идеально подходит для хранения и многолетнего использования
  • Годовая индивидуальная лицензия для MATLAB и Simulink
  • Специальный онлайн-контент для преподавателей

Как устроен курс Arduino Engineering Kit?

Содержание курса Arduino Engineering Kit разделено на шесть глав и предназначено для последовательного следования из глав 1-3, которые при необходимости следует использовать в качестве справочного материала. Главы 4, 5 и 6 являются проектами. Они могут быть выполнены в любом порядке, но рекомендуется начинать с главы 4.

Какие языки поддерживает онлайн-платформа?

Электронная платформа Arduino Engineering в настоящее время доступна на английском языке. Другие языки, включая испанский, немецкий и французский, будут добавлены в ближайшее время!

Каковы различия между Arduino Engineering Kit Rev1 и Rev2?

Основные отличия заключаются в том, что в комплект Rev2 входят:

  • Оптимизированный компактный аппаратный дизайн с новым профессиональным Arduino Nano 33 IoT
  • Улучшенное управление батареями с помощью пользовательской платы Nano Motor Carrier
  • Улучшенная коробка и комплектация для лучшего управления классной комнатой и безопасности груза
  • Улучшенная производительность робота с новыми моделями Simulink
  • Сокращенная сборка время и улучшенная общая производительность с улучшенными конструкциями механической структуры
  • Контент обновляется и доступен на новой онлайн-платформе для лучшего взаимодействия с пользователем
  • Обновлен Matlab и Simulink до версии 2020A

Сколько дополнений я должен загрузить после установки MATLAB?

После завершения установки вам нужно будет установить пять надстроек для использования в проектах.

Включены ли MATLAB и Simulink или мне нужно доплачивать?

MATLAB, Simulink, Stateflow и десять других дополнительных продуктов включены в комплект без дополнительной оплаты. В комплект Arduino Engineering Kit входит годовая пользовательская лицензия на эти продукты для одного человека. Эта лицензия предназначена только для образовательных целей, а не для государственных, исследовательских, коммерческих или других организационных целей.
