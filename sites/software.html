Программирование ARDUINO

Язык программирования устройств Ардуино основан на C/C++. Он прост в освоении, и на данный момент Arduino — это, пожалуй, самый удобный способ программирования устройств на микроконтроллерах.

Функции

Главные функции

setup()

Функция setup() вызывается, когда стартует скетч. Используется для инициализации переменных, определения режимов работы выводов, запуска используемых библиотек и т.д. Функция setup запускает только один раз, после каждой подачи питания или сброса платы Arduino.

Принимаемые значения

Нет

Возвращаемые значения

Нет

loop()

После вызова функции setup(), которая инициализирует и устанавливает первоначальные значения, функция loop() делает точь-в-точь то, что означает её название, и крутится в цикле, позволяя вашей программе совершать вычисления и реагировать на них. Используйте её для активного управления платой Arduino.

Принимаемые значения

Нет

Возвращаемые значения

Нет

Цифровой ввод/вывод

pinMode(pin, mode)

Устанавливает режим работы заданного вход/выхода(pin)

Принимаемые значения

pin: номер вход/выхода(pin), который Вы хотите установить
mode: режим одно из двух значение - INPUT или OUTPUT, устанавливает на вход или выход соответственно.

Возвращаемые значения

Нет

digitalWrite(pin, value)

Подает HIGH или LOW значение на цифровой вход/выход (pin)

Принимаемые значения

pin - номер вход/выхода(pin)
value: значение HIGH или LOW

Возвращаемые значения

Нет

digitalRead(pin)

Функция возвращает значение с заданного входа

Принимаемые значения

pin: номер вход/выхода(pin) который Вы хотите считать

Возвращаемые значения

HIGH/LOW

Аналоговый ввод/выввод

analogRead(pin)

Функция возвращает значение с указанного аналогового входа

Принимаемые значения

pin: номер порта аналогового входа с которого будет производиться считывание (A0..A5 для большинства плат, 0..7 для Mini и Nano и 0..15 для Mega)

Возвращаемые значения

0..1024

analogReference(type)

Функция определяет опорное напряжение относительно которого происходят аналоговые измерения

Принимаемые значения

type: определяет используемое опорное напряжение (DEFAULT, INTERNAL или EXTERNAL)

Возвращаемые значения

Нет

analogWrite(pin, value)

Выдает аналоговую величину (ШИМ волну) на порт вход/выхода

Принимаемые значения

pin: порт вход/выхода на который подаем ШИМ сигнал
value: период рабочего цикла значение между 0 (полностью выключено) and 255 (сигнал подан постоянно)

Возвращаемые значения

Нет

Дополнительные функции ввода/вывода

tone(pin,frequency,duration)

Генерирует на порту вход/выхода сигнал — прямоугольную "волну", заданной частоты и с 50% рабочим циклом. Длительность может быть задана параметром, в противном случае сигнал генерируется пока не будет вызвана функция noTone()

Принимаемые значения

pin: номер порта вход/выхода, на котором будет генерироваться сигнал
frequency: частота сигнала в Герцах
duration: длительность сигнала в миллисекундах

Возвращаемые значения

Нет

noTone(pin)

Останавливает сигнал, генерируемый на порту вход/выхода вызывом функции Tone()

Принимаемые значения

pin: номер порта вход/выхода, на котором прекращаем сигнал

Возвращаемые значения

Нет

shiftOut(dataPin, clockPin, bitOrder, value)

Выводит байт информации на порт вход/выхода последовательно (побитно). Вывод может осуществляться как с первого (левого), так и с последнего (правого) бита.

Принимаемые значения

dataPin: номер порта вход/выхода, на который выводятся биты (int)
clockPin: номер порта по которому производится синхронизация (int)
bitOrder: используемая последовательность вывода бит. MSBFIRST (Most Significant Bit First) — слева или LSBFIRST (Least Significant Bit First) — справа.
value: значение (байт) для вывода (byte)

Возвращаемые значения

Нет

pulseIn(pin, value, timeout)

Считывает длину сигнала на заданном порту (HIGH или LOW). Например, если задано считывание HIGH функцией pulseIn(), функция ожидает пока на заданном порту не появиться HIGH.

Принимаемые значения

pin: номер порта вход/выхода, на котором будет ожидаться сигнал. (int)
value: тип ожидаемого сигнала — HIGH или LOW.
timeout (опционально): время ожидания сигнала (таймаут) в микросекундах; по умолчанию - одна секунда. (unsigned long)

Возвращаемые значения

Длина сигнала в микросекундах или 0, если сигнал не получен до истечения таймаута.

Операторы

Управляющие операторы

if..else

Конструкция ветвление if..else при условии = true, выполняется код в блоке if, иначе в else

for

Конструкция for используется для повторения блока операторов, заключенных в фигурные скобки. Счетчик приращений обычно используется для приращения и завершения цикла. Оператор for подходит для любых повторяющихся действий и часто используется в сочетании с массивами коллекций данных/выводов.

switch..case

Подобно конструкции if, switch...case управляет процессом выполнения программы, позволяя программисту задавать альтернативный код, который будет выполняться при разных условиях. В частности, оператор switch сравнивает значение переменной со значением, определенном в операторах case. Когда найден оператор case, значение которого равно значению переменной, выполняется программный код в этом операторе.
* Ключевое слово break является командой выхода из оператора case и обычно используется в конце каждого case

return

Прекращает вычисления в функции и возвращает значение из прерванной функции в вызывающую, если это нужно

break

Break используется для принудительного выхода из циклов do, for или while, не дожидаясь завершения цикла по условию.

continue

Оператор continue пропускает оставшиеся операторы в текущем шаге цикла.

while

While будет вычислять в цикле непрерывно и бесконечно до тех пор, пока выражение в круглых скобках, () не станет равно логическому ЛОЖНО.

do..while

Цикл do работает так же, как и цикл while, за исключением того, что условие проверяется в конце цикла, таким образом, цикл do будет всегда выполняться хотя бы раз.

Данные

Константы

HIGH | LOW

Задание значение на входа/выходе (HIGH - Высокий сигнал, LOW - Низкий сигнал)

INPUT | OUTPUT

Цифровые порты вход/выхода, могут быть установлены в режимы работы как вход или выход: INPUT или OUTPUT. Установка производится функцией pinMode().

true | false

В языке Ардуино есть две константы для представления логических значений истина и ложь: true и false.

Пользовательские константы

Ключевое слово const указывает, что значение переменной является константой и сообщает компилятору о том, что программист не сможет его изменить.

Синтаксис

const 'тип константы' 'название константы' = 'значение'
